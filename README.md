# DevCamper API

> Backend API for DevCamper App (Bootcamp directory website)

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own.

## Install Dependencies

```
npm install
```

## Run App

```
# Dev Mode
npm run dev

#Prod Mode
npm start
```
